# Generated code

Files under this directory are generated from protobuf schemas and should not be edited by hand.

Regenerate the chatapp TypeScript bindings from the Pinocchio repository root with:

```bash
buf generate --template buf.chatapp.web.gen.yaml --path proto/pinocchio
```

The matching Go bindings are generated by `buf.chatapp.gen.yaml` under `pkg/chatapp/pb`.
